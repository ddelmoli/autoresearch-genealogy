# Verify An AI Finding

Use this before accepting any AI-suggested fact.

## Source Check

- [ ] Did the AI provide a source name, URL, image, or citation?
- [ ] Did you open the source yourself?
- [ ] Does the source actually say what the AI claims?
- [ ] Is the source original, indexed, transcribed, or copied from another tree?
- [ ] Did you record the source in `Research_Log.md` or the person file?

## Identity Check

- [ ] Does the name match, including plausible spelling variants?
- [ ] Do the dates fit the known life timeline?
- [ ] Does the place fit the family geography?
- [ ] Do spouse, parents, children, occupation, religion, or witnesses support the match?
- [ ] Could another same-name person explain the record?

## Decision

- [ ] Accept only if source, identity, date, place, and relationship all make sense.
- [ ] Mark as `speculative` if it is useful but not proven.
- [ ] Reject if the source is missing, unrelated, or copied from an unsourced tree.
- [ ] Log the decision and why.
- [ ] Add a follow-up to `Open_Questions.md` if more evidence is needed.
