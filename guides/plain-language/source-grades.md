# Source Grades In Plain Language

Use this when you need to decide how much to trust a source.

## The Simple Scale

| Plain phrase | Formal tier | What it means |
|---|---|---|
| Trust this | `strong_signal` | A strong source supports the claim, or independent sources agree with no major conflict |
| Use carefully | `moderate_signal` | The source is useful, but the claim still needs more support or has mild uncertainty |
| Lead only | `speculative` | The source points somewhere to search, but it should not be treated as fact |

## Trust This

Use `strong_signal` when:

- A primary record clearly states the fact.
- Two independent sources agree.
- The person, date, place, and relationship all fit.
- No better source contradicts it.

Fictional example:

An official marriage record says Morgan Hale married Riley Stone in 1912, and a church register shows the same couple and date. That marriage can usually be treated as strong unless another source creates a serious conflict.

## Use Carefully

Use `moderate_signal` when:

- One good source supports the claim, but no second source has been found.
- Several secondary sources agree, but they may depend on the same original.
- The record probably fits the person, but one detail is fuzzy.

Fictional example:

An obituary says Lee Parker was born in River County, but no birth record has been found. Use the birthplace carefully and keep looking.

## Lead Only

Use `speculative` when:

- The claim comes from one copied tree.
- The claim comes from family memory only.
- The source is a hint, index, photo label, or unsourced note.
- The same-name match could be another person.

Fictional example:

A user tree says Avery Brooks's father was Nolan Brooks, but gives no source. That is a lead. Search for records before adding Nolan as a parent.

## Fast Decision

- [ ] Did I open the source myself?
- [ ] Does it name the right person?
- [ ] Does the date and place fit?
- [ ] Does it state the relationship I want to add?
- [ ] Is there a second independent source?

If the answer is mostly yes, it may be strong. If the answer is partly yes, use carefully. If the answer is mostly no, lead only.
