# Examples

Worked examples showing autoresearch genealogy in action. All examples use placeholder names and anonymized data. The research patterns and decision-making are real; the family details are not.

## Examples

| Example | Demonstrates |
|---|---|
| [tree-expansion-session.md](tree-expansion-session.md) | How a tree expansion prompt discovers new ancestors over 8 iterations |
| [cross-reference-audit.md](cross-reference-audit.md) | How the audit prompt finds and resolves discrepancies |
| [dna-to-genealogy-mapping.md](dna-to-genealogy-mapping.md) | How to connect DNA test results to documented ancestor lines |
| [name-resolution.md](name-resolution.md) | How to resolve a single ancestor appearing under multiple names |
| [colonial-deep-dive.md](colonial-deep-dive.md) | How to research a colonial American ancestor with limited records |

## How to Read These Examples

Each example shows:
1. The starting state (what was known before)
2. The prompt or approach used
3. The search process (including negative results)
4. The outcome and what was learned
5. What changed in the vault as a result
