# First Week Checklist

Use this before your first autonomous prompt.

## Set Up

- [ ] Copy `vault-template/` into a new genealogy folder.
- [ ] Open `Family_Tree.md`.
- [ ] Add only people and facts you already know.
- [ ] Mark living people as `Living` or `life_status: living`.
- [ ] Leave unknown dates blank instead of guessing.

## Source What You Know

- [ ] Add a source note beside every fact you can explain.
- [ ] Label family memory as family memory.
- [ ] List every document, photo, letter, or certificate in `Data_Inventory.md`.
- [ ] Add the biggest unknowns to `Open_Questions.md`.
- [ ] Add searched-but-not-found notes to `Research_Log.md` when relevant.

## Before AI

- [ ] Remove exact private details for living people from anything you will paste into a public AI tool.
- [ ] Read [Getting Started](../workflows/getting-started.md).
- [ ] Choose one next action with [Prompt Picker](../guides/prompt-picker.md).
- [ ] If you already have a tree, run [02 Cross-Reference Audit](../prompts/02-cross-reference-audit.md) before expansion.

## Done When

- [ ] Your folder has `Family_Tree.md`, `Research_Log.md`, `Open_Questions.md`, and `Data_Inventory.md`.
- [ ] Living people are protected.
- [ ] You know which facts are sourced, remembered, or unknown.
- [ ] You have one next prompt or bundle, not five.
